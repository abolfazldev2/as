# فاز ۱ – پیکربندی شبکه (Network Configuration)

## معرفی

فاز ۱ یک ابزار تعاملی مبتنی بر ترمینال (TUI) برای مدیریت تنظیمات پایه شبکه در سیستم‌عامل لینوکس است.  
هدف این فاز، ساده‌سازی عملیات رایج شبکه بدون نیاز به ویرایش دستی فایل‌ها یا اجرای مستقیم دستورات پیچیده می‌باشد.

کاربر می‌تواند از طریق منوهای گرافیکی در ترمینال، تنظیمات شبکه را به‌صورت مرحله‌به‌مرحله و ایمن اعمال کند.

این فاز روی **Debian 12** تست شده و از **NetworkManager** به‌عنوان backend استفاده می‌کند.

---

## قابلیت‌ها

در فاز ۱ امکانات زیر ارائه می‌شود:

### تنظیمات DNS
- تنظیم DNS موقت (Runtime)
- تنظیم DNS دائمی (Persistent)

### تنظیمات IP
- فعال‌سازی DHCP به‌صورت تمیز
- تنظیم IP استاتیک موقت
- تنظیم IP استاتیک دائمی (با گیت‌وی اختیاری)

### مدیریت Route
- افزودن Route موقت
- افزودن Route دائمی
- حذف Route موقت
- حذف Route دائمی

### تنظیم Hostname
- تغییر نام میزبان سیستم به‌صورت دائمی

تمام ورودی‌های کاربر قبل از اعمال تغییرات بررسی (Validation) می‌شوند تا از ورود مقادیر اشتباه جلوگیری شود.

---

## پیش‌نیازها

قبل از استفاده از این ابزار، موارد زیر باید روی سیستم موجود باشد:

- سیستم‌عامل لینوکس (تست‌شده روی Debian 12)
- Python نسخه 3.10 یا بالاتر
- نصب بودن NetworkManager
- دسترسی root (برای اعمال تنظیمات شبکه)

ابزارهای سیستمی مورد نیاز:
- `nmcli`
- `ip`
- `resolvectl`
- `hostnamectl`
